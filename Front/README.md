# توثيق الواجهة الأمامية (Frontend Documentation) - EVO

هذا المجلد يحتوي على كود الواجهة الأمامية لمشروع **EVO**، والمبني باستخدام إطار العمل **Next.js**.

## التقنيات المستخدمة
- **Next.js 14:** للإطارات والراوتينج (App Router).
- **React 18:** لبناء المكونات التفاعلية.
- **Tailwind CSS:** لتنسيق الواجهات بشكل عصري وسريع.
- **Recharts:** لعرض الرسوم البيانية والإحصائيات.

## هيكل الصفحات (Pages Structure)

تعتمد الصفحات على نظام `App Router` في Next.js وتوجد داخل المجلد `app/`:

### 1. الصفحة الرئيسية (`app/page.js`)
هي صفحة الهبوط (Landing Page) الخاصة بالجامعة، وتشمل:
- **Hero Section:** تعريف بالجامعة ورؤيتها.
- **Departments:** عرض للأقسام المتاحة (Data Science, AI, Cybersecurity, etc.).
- **News Section:** آخر أخبار الجامعة مع نظام عرض (Carousel) تفاعلي.
- **Circular Menu:** قائمة دائرية عائمة للتنقل السريع.

### 2. صفحة تسجيل الدخول (`app/(pages)/login/page.js`)
- تتيح للمستخدمين (طلاب/دكاترة/أدمن) الدخول إلى حساباتهم.
- تحتوي على وظائف التحقق من البيانات (`handleSubmit`) والتعامل مع حالات النجاح أو الخطأ.

### 3. بوابة الطالب (`app/(pages)/student-page/page.js`)
لوحة تحكم متكاملة للطالب تحتوي على المكونات التالية:
- **Profile:** عرض بيانات الطالب الشخصية (الاسم، التخصص، الكود).
- **Grades:** عرض الدرجات والنتائج.
- **Lectures:** جدول المحاضرات والمواد.
- **Statistics:** رسوم بيانية توضح مستوى الطالب وأدائه.
- **Assignments:** المهام المطلوبة ومواعيد تسليمها.
- **Roadmap:** الخطة الدراسية للطالب.
- **Payments:** الموقف المالي والمصروفات.

### 4. القسم الإداري والتحكم (`app/(pages)/control`)
- مخصص لإدارة شؤون الطلاب والكنترول.

### 5. صفحات إضافية:
- **Documents:** لإدارة ورفع الملفات والوثائق.
- **FAQ:** الأسئلة الشائعة وإجاباتها.
- **GPS:** خريطة الجامعة وتحديد المواقع داخل الحرم الجامعي.
- **Library:** نظام المكتبة الرقمية.
- **News:** صفحة مفصلة لأخبار الجامعة.

## المكونات الرئيسية (Common Components)
- **Header:** شريط التنقل العلوي ويظهر في أغلب الصفحات.
- **CircularMenu:** قائمة تفاعلية تظهر في أسفل الصفحة للتنقل بين الأقسام الرئيسية.
- **Icons:** مجموعة من الأيقونات المخصصة المستخدمة في النظام.

## الوظائف البرمجية المهمة
- **Authentication Flow:** يتم التعامل مع تسجيل الدخول وإدارة الـ Token (نخطط لربطه مع Backend بالكامل).
- **Responsive Design:** جميع الصفحات مصممة لتعمل بكفاءة على جميع أحجام الشاشات (Mobile & Desktop) باستخدام Tailwind CSS و Media Queries مخصصة.
- **Real-time Filtering:** وظائف البحث والفلترة في صفحات الأخبار والمكتبة.
